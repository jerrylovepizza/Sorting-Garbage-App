# React-Auth
* React app with passport authentication

[Check out the app here](https://react-auth-simple.herokuapp.com/)

## Requirements
* MongoDB
* Npm or Yarn
* Node
* Terminal or bash

## Built With:
- [x]  React.js
- [x]  passport.js
- [x]  Express
- [x]  mongodb
- [x]  mongoose
- [x]  Node
- [x]  Reactstrap
- [x]  Animate CSS

## Setup Instructions
1. Clone the repo 
    * `git clone git@github.com:evanejtucker/react-auth.git`
    * `cd react-auth`
2. Install dependencies
    * `npm run install`
3. Start mongo server by opening a seperate terminal or bash window and run:
    * `mongod`
4. Start the development server
    * `npm run start`
5. Happy Hacking!
