## react-auth
react app with passport authentication

[Check out the app here](https://react-auth-simple.herokuapp.com/)

### Requirements
* mongoDB
* npm or yarn
* Node
* Terminal or bash

### built with:
- [x]  React.js
- [x]  passport.js
- [x]  Express
- [x]  mongodb
- [x]  mongoose
- [x]  Node
- [x]  reactstrap
- [x]  animate css

### Setup Instructions
1. clone the repo 
    * `git clone git@github.com:evanejtucker/react-auth.git`
    * `cd react-auth`
2. install dependencies
    * `npm run install`
3. Start mongo server by opening a seperate terminal or bash window and run:
    * `mongod`
4. Start the development server
    * `npm run start`
5. Happy Hacking!
